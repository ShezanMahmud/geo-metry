1. What is DOM and what is the purpose of DOM?

Answer-1 : The DOM (Document Object Model) is an interface that represents how your HTML and XML documents are read by the browser.
It allows a language (JavaScript) to manipulate, structure, and style your website

2. How will you select HTML elements using DOM? Name the DOM methods.

Answer-2 : 5 ways in which we can select elements in a DOM using selectors.

getElementsByTagName(), getElementsByClassName(), getElementById(), querySelector(), querySelectorAll()

3. What is event bubble?

Answer-3 : Event bubbles are a type of event propagation in JavaScript.he bubbles event property returns true if an event is a bubbling event.

4. What is a callback function and why will you use it?

Answer-4 : A callback function is a function that is passed as an argument to another function, which is then called inside the outer function to complete some routine or action.
